Expandable Floating Action Button

A CSS-snapped floating speed dial component. Accessible, keyboard focusable, support for click triggers, CSS hover variants, and staggered transitions.

Theme

Click Trigger (Up)

Active Overlay

Click the primary blue button in the bottom right corner to fan out actions vertically.

Features a blur backdrop filter overlay that dims background elements and closes on outside click or Esc.

Click Trigger (Right)

Horizontal Dial

Click the primary button in the bottom left to fan out actions horizontally along the bottom margin.

Tooltips align on the right relative to the actions dynamically.

Hover Trigger (Up)

Pure CSS Trigger

Hover the button in the bottom right corner. The actions fan out immediately with staggered delays without clicking.

Uses .ease-speed-dial--hover. Pauses actions when cursor leaves the boundary.

Part of the EaseMotion CSS Library
